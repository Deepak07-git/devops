# DevOps CI/CD Pipeline for Flask Application
Complete DevOps project using Flask, Docker, Jenkins, Kubernetes & Terraform.
This project is a complete end-to-end DevOps pipeline built using a Python Flask web application.
It demonstrates CI/CD automation, containerization, Kubernetes deployment, and Infrastructure as Code using Terraform.

This project showcases:

Python Flask Application

Docker Containerization

Jenkins CI/CD Pipeline

Kubernetes Deployment (Deployment + Service)

Terraform for AWS Infrastructure

GitHub for Version Control

DockerHub for Image Registry
